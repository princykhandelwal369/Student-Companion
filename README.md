# Student Companion

A web application built with Flask to help students organize their academic life in one place.

> This project is currently under active development as part of my Summer 2026 learning roadmap. I am learning Flask from scratch while building this application, documenting my progress and improving it feature by feature.

---

## About the Project

Student Companion aims to become a productivity tool for students by bringing together essential academic features in a simple and clean interface.

Instead of building several small practice projects, I chose to learn Flask by developing one application from the ground up. As I learn new concepts, they are directly integrated into this project.
